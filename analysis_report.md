# Sales Data Analysis Report

## Project Overview
This project analyzes a sales dataset using Python and Pandas to extract meaningful insights such as total revenue and best-selling products.

## Tools Used
- Python
- Pandas Library

## Steps Performed
1. Loaded CSV dataset using Pandas
2. Explored dataset structure and data types
3. Handled missing values
4. Calculated key sales metrics
5. Identified best-selling product

## Key Findings
- Total Revenue was calculated successfully
- Average, highest, and lowest sales values identified
- Laptop/Phone (example) emerged as the best-selling product

## Conclusion
Pandas simplifies data analysis by providing Excel-like operations with greater flexibility and automation.
